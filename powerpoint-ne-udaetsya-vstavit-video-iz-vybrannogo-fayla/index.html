PowerPoint не удается вставить видео из выбранного файла

Время чтения 4 мин.Опубликовано 21.06.2024

Установите пакет кодеков K-Lite, чтобы устранить проблему.

  • Чтобы исправить это, вам необходимо использовать несколько решений, таких как загрузка 64-разрядного файла кодека.

Предположим, вы получаете PowerPoint и не можете вставить видео из выбранного файла, проверьте необходимый 64-разрядный кодек при попытке добавить видео на слайд PowerPoint.

Из-за чего PowerPoint не может вставить видео из выбранного файла?

Могут быть различные причины, по которым PowerPoint не может вставить видео из выбранного файла; некоторые из популярных:

  • Формат видео не поддерживается – PowerPoint поддерживает ограниченное количество форматов видеофайлов, включая MP4, AVI и WMV. При попытке добавить в PowerPoint видеофайл, формат файла которого не поддерживается, может возникнуть эта ошибка.
  • Несовместимый видеокодек – если видеокодек, используемый для сжатия видео, не распознается PowerPoint, то вы можете получить эту ошибку.
  • Поврежден видеофайл – Если видеофайл, который вы пытаетесь добавить в PowerPoint, поврежден, вы не сможете его добавить, отсюда и проблема.
  • Видеофайл большого размера – в PowerPoint установлен предельный размер файла для видео. Поэтому, если вы добавляете видеофайл большого размера, который PowerPoint не может обработать, может возникнуть эта ошибка.
  • Отсутствующие кодеки – если на вашем компьютере отсутствуют необходимые кодеки для PowerPoint, вы можете увидеть эту ошибку. Вам необходимо установить кодеки самостоятельно.
  • Видеофайл защищен – Видеофайл, который вы пытаетесь добавить в PowerPoint, защищен или встроен. Поэтому вам нужны права доступа к видеофайлу.

Что я могу сделать, если PowerPoint не может вставить видео из выбранного файла?

Прежде чем приступить к устранению неполадок, выполните следующие предварительные проверки, чтобы устранить проблему:

  • Используйте файлы .MP4, закодированные в видео H.264 и аудио AAC. Видео будет совместимо с PowerPoint как для Windows, так и для Mac.
  • Проверьте видеофайл, чтобы убедиться, что он был загружен или скопирован правильно и не был поврежден.
  • Сожмите видео, если это файл большого размера.

1. Установите K-Lite Codec Pack

  1. Перейдите на официальный веб-сайт пакетов кодеков K-Lite.
  2. Вы получите четыре варианта пакета. Сначала выберите стандартный пакет и загрузите его.
  3. При появлении запроса выберите Сохранить.
  4. Дважды щелкните на K-Lite_Codec_Pack_1365_Standard.exe и выберите соответствующий параметр в мастере установки.
  5. Выберите обычный режим.
  6. Нажмите Далее.
  7. Измените или примите настройки установки, затем нажмите Далее.
  8. В дополнительных задачах и параметрах нажмите Далее.
  9. Нажмите Далее в окне настройки MPC-HC.
  10. Установите флажок рядом с Методом декодирования в разделе Аппаратное ускорение. Нажмите Далее.
  11. На следующей странице для предпочитаемого языка примите значение по умолчанию и нажмите Далее.
  12. Нажмите Далее в настройках конфигурации звука. В следующем окне вам будет предложено дополнительное программное обеспечение; отклоните предложение.
  13. Теперь в разделе «Готово к установке» нажмите кнопку «Установить».
  14. После установки перезагрузите компьютер, затем попробуйте добавить видео в PowerPoint.

2. Измените формат видео

  1. Откройте предпочитаемый браузер и перейдите на веб-сайт online video Converter.
  2. Нажмите на кнопку Выбрать или удалить файл на веб-сайте, чтобы добавить свой видеофайл.
  3. В списке различных видеоформатов выберите WMV.
  4. Веб-сайт конвертирует видеофайл, и вы сможете загрузить его, как только он будет готов.

3. Внесите изменения в настройки видео.

  1. Выберите, щелкните правой кнопкой мыши видеофайл и выберите Свойства.
  2. Теперь на вкладке «Общие» поставьте галочку рядом с «Разблокировать».
  3. Нажмите Применить и OK. Попробуйте добавить видео еще раз.

4. Включите элементы управления проигрывателем Windows Media в PowerPoint

  1. Запустите PowerPoint. Щелкните Файл.
  2. Выберите Параметры.
  3. Нажмите Настроить ленту; из команд выбора здесь выберите Основные вкладки.
  4. Выберите Разработчик из списка и выберите Добавить.
  5. Нажмите OK.
  6. Теперь на вкладке «Разработчик» выберите «Дополнительные элементы управления» в разделе «Элементы управления».
  7. В следующем окне выберите проигрыватель Windows Media и нажмите OK. Попробуйте добавить видео в PowerPoint.

5. Обновите драйверы

  1. Нажмите Windows + R, чтобы открыть окно Выполнить.
  2. Введите devmgmt.msc и нажмите Enter, чтобы открыть диспетчер устройств.
  3. Найдите и щелкните Звуковые, видео- и игровые контроллеры, чтобы развернуть его.
  4. Щелкните правой кнопкой мыши по каждому имеющемуся драйверу по очереди и выберите Обновить драйвер.
  5. В следующем окне выберите Автоматический поиск драйверов.
  6. Для завершения следуйте инструкциям на экране.
