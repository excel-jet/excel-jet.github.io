Функция CONVERT в Excel

Время чтения 5 мин.Опубликовано 03.10.2024

Функция CONVERT (ПРЕОБР) Excel преобразует число из одной системы измерения в другую. Например, вы можете использовать CONVERT для преобразования футов в метры, фунтов в килограммы, градусов по Фаренгейту в градусы Цельсия, галлонов в литры и для многих других преобразований единиц измерения.

Что делает функция CONVERT Excel?

Функция CONVERT Excel преобразует единицы измерения.

Возвращаемое значение

Число в новой системе измерения.

Аргументы

  • number — числовое значение для преобразования.
  • from_unit — начальные единицы измерения числа.
  • to_unit — конечные единицы результата.

Синтаксис

=CONVERT(number, from_unit, to_unit)

Примечания по использованию

Функция ПРЕОБР преобразует число из одной системы измерения в другую. Например, вы можете использовать CONVERT для преобразования футов в метры, фунтов в килограммы, галлонов в литры и для многих других преобразований единиц измерения.

Примеры

В приведенных ниже формулах используется функция ПРЕОБР для преобразования ярдов в метры, градусов Цельсия в градусы Фаренгейта, галлонов в литры и квадратных метров в квадратные ярды:

=CONVERT(100,"yd","m") // возвращает 91.44
=CONVERT(22,"C","F") // возвращает 71.6
=CONVERT(1,"gal","l") // возвращает 3.79
=CONVERT(100,"m2","ft2") // возвращает 1076.39

Примечания

  • Функция ПРЕОБР в экселе чувствительна к регистру.
  • CONVERT вернет ошибку #N/A, если строка единицы измерения не распознана.
  • Функция вернет ошибку #N/A, если единицы измерения несовместимы.
  • Функция вернет #VALUE! ошибка, когда номер недействителен.
  • В функцию CONVERT в Excel 2013 было добавлено несколько единиц измерения.
